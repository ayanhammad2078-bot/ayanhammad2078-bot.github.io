# Calculator Hub

A complete, free, multi-page calculator website built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies. Ready to deploy on GitHub Pages in minutes.

## Files Included

| File | Description |
|---|---|
| `index.html` | Homepage with links to all 10 calculators |
| `style.css` | Shared stylesheet for all pages |
| `bmi.html` | BMI Calculator |
| `loan-calculator.html` | Loan / Mortgage Calculator |
| `gpa-calculator.html` | GPA Calculator |
| `percentage-calculator.html` | Percentage Calculator (3 modes) |
| `age-calculator.html` | Age Calculator |
| `compound-interest.html` | Compound Interest Calculator |
| `calorie-calculator.html` | Calorie / TDEE Calculator |
| `unit-converter.html` | Unit Converter (5 categories) |
| `discount-calculator.html` | Discount Calculator (3 modes) |
| `time-calculator.html` | Time Calculator (add/subtract + convert) |

## Deploy to GitHub Pages

1. Create a new GitHub repository (e.g., `calculator-hub`).
2. Upload all files from this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Source**, select `main` branch and `/ (root)` folder.
5. Click **Save**. Your site will be live at `https://yourusername.github.io/calculator-hub/`.

## AdSense Integration

Each page contains three clearly marked ad slot placeholders:

- **Top banner** (728×90) — above the main content
- **Middle ad** (468×60) — between the calculator and the info sections
- **Bottom banner** (728×90) — at the bottom of the page

To activate AdSense, replace the `<div class="ad-slot ...">Advertisement</div>` elements with your AdSense `<ins>` ad unit code.
