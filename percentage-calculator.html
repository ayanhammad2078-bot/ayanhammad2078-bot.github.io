How to Use the Percentage Calculator

  • Mode 1: Enter a percentage and a number to find what that percentage of the number equals (e.g., 25% of 200 = 50).
  • Mode 2: Enter two numbers to find what percentage the first is of the second (e.g., 50 is 25% of 200).
  • Mode 3: Enter an original and a new value to find the percentage increase or decrease between them.

Understanding Percentage Results

A percentage is a way of expressing a number as a fraction of 100. Percentage change shows how much a value has increased or decreased relative to its original value. A positive percentage change means an increase; a negative value means a decrease.

Frequently Asked Questions

How do I calculate a percentage of a number?

Multiply the number by the percentage and divide by 100. For example, 20% of 150 = (20 × 150) / 100 = 30.

How do I find what percentage one number is of another?

Divide the first number by the second and multiply by 100. For example, 30 is what percent of 150? (30 / 150) × 100 = 20%.

How is percentage change calculated?

Percentage change = ((New Value - Original Value) / Original Value) × 100. A positive result is an increase; a negative result is a decrease.

What is the difference between percentage and percentage points?

A percentage is a ratio out of 100. A percentage point is the arithmetic difference between two percentages. For example, going from 10% to 15% is a 5 percentage point increase, but a 50% relative increase.
